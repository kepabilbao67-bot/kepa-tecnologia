## Estrategia de Negocio en 3 Fases

Este repositorio documenta la estrategia de negocio de Kepa Tecnologia organizada en **tres fases claramente diferenciadas**. Cada fase construye sobre la anterior, creando una progresion logica desde generar ingresos vendiendo para otros hasta escalar el negocio propio.

---

## Las 3 Fases

### FASE 1: Comprar en Auvesta y Vender para Ellos
**Objetivo:** Generar ingresos iniciales vendiendo para Auvesta mientras se aprende el sistema.

- Comprar participacion en Auvesta
- Trabajar vendiendo sus productos/servicios
- Usar automatizaciones para optimizar el proceso de venta
- Generar ingresos desde el primer dia

📄 [Ver documentacion completa de Fase 1](./fase-1-auvesta-venta.md)

---

### FASE 2: Vender Servicios y Automatizaciones a los Partners de Auvesta
**Objetivo:** Monetizar las habilidades tecnicas vendiendo servicios a la red de partners.

- Identificar partners de Auvesta que necesiten mejorar su presencia digital
- Ofrecer arreglar/mejorar sus landing pages
- Gestionar sus redes sociales
- Implementar automatizaciones para encontrar clientes usando IA
- Vender servicios de Make.com, Claude API, y captacion automatizada

📄 [Ver documentacion completa de Fase 2](./fase-2-servicios-partners.md)

---

### FASE 3: Aplicar la Misma Estrategia al Negocio Propio (Kepa Tecnologia)
**Objetivo:** Usar todo lo construido para escalar el negocio freelance propio.

- Aplicar las mismas automatizaciones a kepatecnologia.com
- Gestionar propias redes sociales con las herramientas creadas
- Captar clientes propios usando IA y automatizaciones
- Escalar el negocio freelance con sistemas probados

📄 [Ver documentacion completa de Fase 3](./fase-3-kepa-tecnologia.md)
