## Paleta de Colores

| Color | Hex | Uso Principal |
|-------|-----|---------------|
| Naranja (principal) | `#ff6a14` | CTAs, titulares, acentos, elementos destacados |
| Fondo oscuro | `#060606` | Fondo de posts, fondos de stories |
| Texto claro | `#f3eee9` | Texto principal sobre fondo oscuro |
| Cyan | `#00d4ff` | Enlaces, datos tecnicos, elementos secundarios |
| Amarillo | `#ffd400` | Alertas, numeros importantes, badges |
| Gris oscuro | `#1a1a1a` | Fondos de tarjetas, separadores |
| Gris medio | `#333333` | Bordes, lineas divisorias |

### Reglas de Color
- **Fondo siempre oscuro** (#060606 o #1a1a1a). Nunca fondos blancos o claros.
- **Naranja para lo importante**: titulares, botones, elementos que deben destacar.
- **Cyan para datos tecnicos**: numeros, porcentajes, nombres de herramientas.
- **Amarillo con moderacion**: solo para alertas o datos que necesiten maxima atencion.
- **Texto siempre claro** (#f3eee9) sobre fondo oscuro. Nunca texto oscuro.

### Combinaciones Aprobadas
```
✅ Fondo #060606 + Texto #f3eee9 + Titulo #ff6a14
✅ Fondo #1a1a1a + Texto #f3eee9 + Acento #00d4ff
✅ Fondo #060606 + Numero #ffd400 + Texto #f3eee9
❌ Fondo blanco + cualquier cosa
❌ Naranja sobre cyan (contraste malo)
❌ Amarillo sobre fondo claro
```

## Tipografias

| Fuente | Uso | Alternativa en Canva |
|--------|-----|---------------------|
| **Anton** | Titulares, headlines grandes | "Anton" (disponible en Canva) o "Bebas Neue" |
| **Syne** | Subtitulos, display, nombres | "Syne" (disponible en Canva) o "Space Grotesk" |
| **DM Mono** | Cuerpo, datos tecnicos, codigo | "DM Mono" (disponible) o "JetBrains Mono" o "Source Code Pro" |

### Tamanos Recomendados

| Elemento | Tamano (px) | Peso |
|----------|-------------|------|
| Titulo principal | 64-80px | Bold/Black |
| Subtitulo | 36-48px | SemiBold |
| Cuerpo de texto | 24-32px | Regular |
| Caption/small | 16-20px | Regular |
| Datos/numeros destacados | 72-120px | Bold |

### Reglas de Tipografia
- Titulares SIEMPRE en mayusculas con Anton
- Cuerpo de texto en DM Mono (da aspecto tecnico/profesional)
- Maximo 2 fuentes por diseno (titulo + cuerpo)
- Interlineado generoso (1.4x minimo)
- Nunca texto con serifs

## Medidas por Plataforma

| Plataforma | Formato | Medidas (px) | Ratio |
|------------|---------|-------------|-------|
| Instagram Post | Cuadrado | 1080 x 1080 | 1:1 |
| Instagram Story/Reel | Vertical | 1080 x 1920 | 9:16 |
| Instagram Carrusel | Cuadrado | 1080 x 1080 | 1:1 |
| LinkedIn Post | Horizontal | 1200 x 627 | 1.91:1 |
| LinkedIn Carrusel | Vertical | 1080 x 1350 | 4:5 |
| Twitter/X Post | Horizontal | 1200 x 675 | 16:9 |
| TikTok Video | Vertical | 1080 x 1920 | 9:16 |
| YouTube Thumbnail | Horizontal | 1280 x 720 | 16:9 |
| Facebook Post | Horizontal | 1200 x 630 | 1.91:1 |
| WhatsApp Status | Vertical | 1080 x 1920 | 9:16 |

### Zonas Seguras
- **Instagram:** No poner texto importante en los 150px superiores/inferiores (interfaz)
- **Reels/TikTok:** Zona segura central: 1080x1350 dentro del 1080x1920
- **Stories:** No usar los 250px superiores (hora/bateria) ni 200px inferiores (respuesta)

## 5 Plantillas Tipo (Descripciones para Canva)

### Plantilla 1: Post Educativo / Tip

**Estructura:**
```
┌──────────────────────┐
│  [Logo kepa pequeno] │ ← esquina superior izquierda
│                      │
│  TITULO EN GRANDE    │ ← Anton, #ff6a14, 64px
│  (2 lineas max)      │
│                      │
│  ───────────────     │ ← linea naranja separadora
│                      │
│  Cuerpo del tip      │ ← DM Mono, #f3eee9, 28px
│  en 3-4 lineas       │
│  maximo              │
│                      │
│  @kepatecnologia     │ ← esquina inferior derecha, #666
└──────────────────────┘
   Fondo: #060606
```

**Uso:** Tips rapidos, datos, estadisticas, frases clave

---

### Plantilla 2: Carrusel Educativo (Slide)

**Estructura - Portada:**
```
┌──────────────────────┐
│                      │
│                      │
│  TITULO              │ ← Anton, #ff6a14, 72px
│  LLAMATIVO           │
│                      │
│  Subtitulo en        │ ← Syne, #f3eee9, 32px
│  1-2 lineas          │
│                      │
│  → Desliza           │ ← DM Mono, #00d4ff, 24px
│                      │
│  @kepatecnologia     │
└──────────────────────┘
   Fondo: gradiente #060606 → #1a1a1a
```

**Estructura - Slides interiores:**
```
┌──────────────────────┐
│  Numero: 01          │ ← Anton, #ffd400, 48px
│                      │
│  Punto principal     │ ← Syne, #f3eee9, 36px
│                      │
│  Explicacion en      │ ← DM Mono, #f3eee9, 24px
│  texto mas pequeno   │
│  con detalles        │
│                      │
│  [icono relevante]   │ ← color #ff6a14
│                      │
└──────────────────────┘
   Fondo: #060606
```

**Uso:** Tutoriales paso a paso, listas, comparativas

## Checklist de Diseno (Antes de Publicar)

- [ ] Fondo oscuro (#060606 o #1a1a1a)
- [ ] Texto legible (contraste suficiente)
- [ ] Maximo 2 tipografias usadas
- [ ] Colores de marca (naranja, cyan, amarillo)
- [ ] Logo o @kepatecnologia visible
- [ ] Texto dentro de zona segura
- [ ] Medidas correctas para la plataforma
- [ ] Sin faltas de ortografia
- [ ] CTA claro si aplica
- [ ] Vista previa en movil (tamano texto suficiente)
