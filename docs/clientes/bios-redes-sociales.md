## Reglas Generales para Todas las Bios

| Elemento | Recomendacion |
|----------|---------------|
| Nombre | "Kepa Bilbao" o "Kepa Tecnologia" (consistente) |
| Foto de perfil | Logo kepa-logo.png o foto profesional kepa-photo.png |
| Enlace | Siempre kepatecnologia.com (con UTM si se trackea) |
| Ubicacion | Siempre "Bilbao" visible |
| CTA | Siempre incluir una accion clara (visitar web, DM, etc.) |
| Palabras clave | automatizacion, IA, Make.com, PYMEs, Bilbao |

### Variaciones Segun Campana

**Cuando se lanza una oferta:**
- Cambiar ultima linea de bio por la oferta activa
- Ejemplo: "🔥 Gratis: auditoria de automatizacion → link"

**Cuando se busca grupo/equipo:**
- Anadir: "🚀 Unete al equipo → link"
- Cambiar enlace a landing de grupo

**Cuando se promociona contenido:**
- Anadir: "📹 Nuevo video: [tema] → link"
