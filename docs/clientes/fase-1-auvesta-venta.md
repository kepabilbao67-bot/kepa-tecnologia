# FASE 1: Comprar en Auvesta y Vender para Ellos

## Resumen de la Fase

La primera fase consiste en **comprar una participacion en Auvesta** y trabajar vendiendo sus productos/servicios. Esta fase genera ingresos desde el primer dia mientras se construyen las habilidades y sistemas que se usaran en las fases siguientes.

**Objetivo principal:** Generar ingresos iniciales y aprender el sistema de ventas con soporte de automatizaciones.

---

## Que es Auvesta

Auvesta es una empresa que permite a personas comprar una participacion y trabajar como vendedores de sus productos/servicios. Funciona como un modelo donde:

- **Compras** una participacion/licencia para operar
- **Vendes** los productos/servicios de Auvesta a clientes
- **Ganas** comisiones por cada venta realizada
- **Creces** dentro de la red a medida que generas resultados

---

## Como Funciona la Compra

### Pasos para Entrar

1. **Contactar con Auvesta** - Solicitar informacion sobre participacion
2. **Evaluar la inversion** - Entender el coste de entrada y el retorno esperado
3. **Realizar la compra** - Adquirir la participacion/licencia
4. **Recibir formacion** - Aprender los productos y el sistema de ventas
5. **Empezar a vender** - Comenzar a generar ingresos

### Que Incluye la Participacion

- Acceso al catalogo de productos/servicios para vender
- Material de ventas y marketing
- Soporte y formacion
- Acceso a la red de partners
- Herramientas de seguimiento de ventas

---

## Como se Vende

### Proceso de Venta

```
┌─────────────┐    ┌──────────────┐    ┌─────────────┐    ┌──────────────┐
│ Encontrar   │───>│ Clasificar   │───>│ Contactar   │───>│ Cerrar       │
│ leads       │    │ con IA       │    │ segun       │    │ venta        │
│             │    │ (Make.com)   │    │ prioridad   │    │              │
└─────────────┘    └──────────────┘    └─────────────┘    └──────────────┘
```

### Canales para Encontrar Leads

1. **Redes sociales** - Publicaciones, stories, mensajes directos
2. **WhatsApp** - Contactos existentes, grupos relevantes
3. **Referidos** - Personas que recomiendan a otras
4. **Eventos** - Networking presencial en Bilbao
5. **Online** - Formularios web, landing pages

### Clasificacion Automatizada

Aqui es donde entra la automatizacion de Make.com:
- Cada lead que entra se registra en Airtable
- Claude API clasifica automaticamente si es Caliente/Tibio/Frio
- Se asigna la accion siguiente (llamar, email, WhatsApp)
- Se prioriza el tiempo en los leads con mayor probabilidad

Ver: [Documentacion tecnica del flujo Make.com](./automatizacion-make-clasificacion.md)

## Como las Automatizaciones Ayudan en esta Fase

### Problema Sin Automatizacion
- Revisar cada lead manualmente
- No saber a quien llamar primero
- Perder tiempo con leads frios
- Olvidar hacer seguimiento
- No tener datos para mejorar

### Solucion Con Automatizacion

| Tarea | Sin Automatizacion | Con Automatizacion |
|---|---|---|
| Clasificar un lead | 5-10 min manual | 5 seg automatico |
| Decidir proxima accion | Intuicion | Recomendacion IA |
| Seguimiento | Agenda manual | Recordatorios automaticos |
| Reportes | Excel manual | Dashboard en Airtable |
| Tiempo por lead | 15-20 min | 2-3 min |

### Flujo Diario con Automatizaciones

1. **Manana:** Revisar dashboard de Airtable - ver leads clasificados la noche anterior
2. **Priorizar:** Empezar por leads Calientes (puntuacion 7-10)
3. **Contactar:** Seguir la accion recomendada por la IA (llamar/email/WhatsApp)
4. **Registrar:** Anotar resultado en Airtable
5. **Noche:** Los nuevos leads se clasifican solos mientras duermes

## Transicion a Fase 2

Una vez que la Fase 1 esta funcionando y generas ingresos estables:

1. **Identifica partners** que conozcas a traves de Auvesta
2. **Observa sus problemas** - landing pages malas, redes abandonadas, sin automatizaciones
3. **Ofrece soluciones** - Ya tienes las herramientas y la experiencia
4. **Empieza Fase 2** - Vende tus servicios a esos partners

La Fase 1 te da:
- Ingresos para sostener el negocio
- Contactos (partners) para la Fase 2
- Experiencia real con automatizaciones
- Un sistema probado que puedes replicar
